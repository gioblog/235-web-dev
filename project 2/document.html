About The Project

I had an extremely hard time implementing the episodes filter because the episodes is its own API within the Rick and Morty API. So, I was trying to load through an API within an API. After spending weeks on this issue and inhibiting my progress, I opted to exclude it from my final project to save my sanity. I made the results display in a poloroid style. When you hover over each result entry, a shadow will appear underneath I color picked Pickle Rick to use it for the colors of the search button. The three controls in my project is:

  • Filter selector that allows the user to choose if they see the life status, origin, and/or the species.
  • Typing in a term (character) to search for
  • Choose if you want characters with no image to display
